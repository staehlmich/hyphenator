ℹ️ How it works: This tool uses proper German hyphenation patterns (TeX/LaTeX-based) to insert soft hyphens (&shy; or Unicode U+00AD) into words. These invisible characters tell browsers where they can break words if needed. The hyphens only appear when a line break occurs at that position.
